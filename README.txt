# Консольная программа для работы с файлом

Программа позволяет записывать текст в файл, читать его содержимое и считать количество строк и слов через командную строку.

## Установка

1. Сохраните файл 5.py
2. Убедитесь, что установлен Python 3.14
3. Запускайте из командной строки или терминала.

## Использование

### Базовое использование программы

python 5.py [текст для записи] [опции]

Текст принимается на русском языке и на английском. Если текст написан не будет, ничего записано не будет, но и ошибки не будет.
Текст записывается в файл Task1_output.txt .

### Список опций(флагов)

-f FILE или --file FILE - указание имени файла, в который будет записан текст, а также использование с другими опциями
-r или --read - чтение содержимого указанного файла
-c или --count - подсчёт количества строк и слов в указанном файле

## Примеры использования

### Запись текста в файл по умолчанию

python 5.py Привет мир
Текст "Привет мир" будет записан в файл Task1_output.txt .

### Запись текста в указанный файл

python 5.py Привет мир --file my file
Текст "Привет мир" будет записан в файл my_file.txt .

### Чтение текста в файле

python 5.py --file my_file --read
Таким образом будет выведено содержание файла my_file.txt построчно.

python 5.py --read
Будет выведено содержание файла Task1_output.txt построчно.

### Подсчёт количества строк и слов в файле.

python 5.py --count
Будет выведено количество строк и слов в файле Task1_output.txt .

python 5.py --file my file --count
Будет выведено количество строк и слов в файле my_file.txt .

### Комбинирование опций.

python 5.py Текст для записи в файл --file my file --count --read
Сперва текст будет записан в файл my_file.txt, затем будет подсчитано и выведено количество строк и
слов в файле, и наконец будет выведено содержимое файла.

## Особенности работы.

1. Если не указывать файл через -f или --file, используется Task1_output .
2. Текст всегда добавляется в конец файла.
3. Имена файлов автоматически получают расширение .txt, если его нет.
4. Из имени файла удаляются специальные символы, а пробелы заменяются на нижние подчёркивания.
5. Программа выводит какой-либо текст только при использовании опций --count или --read
